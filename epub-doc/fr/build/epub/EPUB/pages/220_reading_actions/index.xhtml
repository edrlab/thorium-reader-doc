Rechercher dans un livre

Vous pouvez utiliser les touches Ctrl + F pour accéder au champ de recherche. Entrez votre terme de recherche. En cliquant sur le bouton Search , vous obtenez les boutons Back et Next pour naviguer entre les occurrences. Le nombre de correspondances trouvées est un lien qui ouvrira une List of results dans le panneau de droite. Chaque correspondance affichée est un lien que vous pouvez Activer pour vous rendre à l'endroit du livre où l'expression recherchée a été trouvée.

Attention : n'oubliez pas qu'en effectuant une recherche et en parcourant les résultats, vous perdrez le point de lecture où vous étiez. Placez simplement un signet pour vous assurer de retrouver votre chemin.
