Marquer des pages

Ajouter un marque page

Le bouton marque page vous permet d'ajouter un marque page personnalisé. Ce peut être un point fixe ou une sélection de texte. Il ne peut y avoir qu'un seul signet à point fixe par page (pages du livre ou pages calculées dans le cas où il n'y a pas de référence de page dans le livre). En revanche, vous pouvez placer autant de marque page de sélection de texte que vous le souhaitez.

Lorsque vous ajoutez un marque page, une notification apparaît pour vous informer que le marque page a été Ajouté ou Supprimé. Le nombre qui suit indique le nombre total de marque page dans le Livre après cette action.

Sélectionnez le texte à marquer. Appuyez sur le bouton marque page. ou la combinaison de touches Ctrl + B.

Si aucun texte n'est sélectionné, le dernier emplacement de lecture connu sera marqué.

Révision et navigation dans les marque page

Appuyez sur le bouton Navigation ( Ctrl + N), puis sélectionnez marque page.Vous pouvez modifier le nom de chaque marque page, supprimer des marque page et passer d'un marque page à l'autre en appuyant sur Entrée.

Accédez à la liste des marque page dans la zone de navigation. Sélectionnez le signet auquel vous souhaitez accéder et appuyez sur la touche Enter. Si vous avez renommé le signet, il vous sera facile d'accéder à plusieurs endroits du livre.

Si vous sélectionnez une portion de texte et activez ensuite le bouton marque page, le signet créé aura pour titre la ligne sélectionnée.

Si vous avez créé un marque page sur une ligne sans sélectionner de portion de texte, essayez de copier la ligne sur laquelle vous vous trouvez dans le presse-papiers. Allez ensuite dans les options de navigation avec Ctrl + N, puis faites défiler vers le bas jusqu'à la section marque page, où ils sont répertoriés. Appuyez sur la touche b pour accéder au bouton d'édition et renommer le marque page. Collez la ligne du presse-papiers et appuyez sur Entrée. Le marque page est renommé avec la ligne de texte que vous avez copiée. Lorsque vous avez plusieurs signets, ce renommage des signets vous permet de les identifier plus facilement.

Depuis la version 3.2, il est possible d'ajouter des Étiquettes aux signets pour faciliter le filtrage, le regroupement et la recherche d'emplacements spécifiques.
