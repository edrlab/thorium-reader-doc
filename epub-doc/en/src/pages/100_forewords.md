# Welcome to Thorium Reader 

Thorium Reader is a versatile digital book reader and library manager that supports multiple formats including EPUB, PDF, audiobooks, and DAISY publications. This documentation will guide you through all features and capabilities.

This software is free and open-source. Its development is made possible by EDRLab members, public grants, and donations from users like you. If you find Thorium valuable, please consider [supporting the project](https://thorium.edrlab.org/en/th3/900_about_thorium/901_thorium-sponsorship/).

## Supported Formats

Thorium Reader supports various publication formats, each with specific capabilities:

* **EPUB** - The most feature-rich format. EPUB 3 provides the best reading experience with support for accessibility features, multimedia, and interactive content. EPUB 2 files are also supported.
* **DAISY** - Specialized format offering enhanced accessibility. Supports DAISY 3 and 2.02 in text-only, audio-only, and synchronized text-audio modes.
* **PDF** - Fixed-layout format preserving the print appearance. Supports text selection, search, bookmarks, and zoom.
* **Audiobooks** - W3C standardized audiobook format (packaged LPF).
* **Visual narratives** - Comics, manga, and image-based content using the Divina format (experimental).

## Touch Gestures

Thorium supports basic touch gestures for tablets and touch-enabled devices:

- **Tap** - Activate buttons and links
- **Tap and drag** - Select text
- **Swipe** - Turn pages (paginated mode) or scroll (continuous mode)
- **Pinch/stretch** - Zoom images
- **Press and hold** - Open context menu

</section>
