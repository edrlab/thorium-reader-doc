Thorium es un lector de libros digitales, también es una biblioteca donde puedes Almacene sus libros y acceso a catálogos en línea.

Thorium permite leer libros electrónicos en diferentes formatos, cada uno de ellos tiene fortalezas y limitaciones. Esta documentación es proporcionado como EPUB, el formato con el mayor número de funcionalidades.

Este software es gratuito, de código abierto y, como tal, su desarrollo es solo posible porque está financiado por diferentes organizaciones. Entre estos, miembros de edrlab, subvenciones privadas o públicas y pequeñas recurrentes donaciones de usuarios felices (individuos o empresas). Si confías en Esta aplicación, considere apoyo con financiación y patrocinio
