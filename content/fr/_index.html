Thorium 3 Guide d'utilisation

La documentation est disponible au format EPUB ; diverses traductions sont disponibles dans le catalogue de documentation de Thorium.

Une fois Thorium Reader téléchargé, installé et lancé une fois, cliquez sur le bouton ci-dessous pour ouvrir le catalogue de documentation dans Thorium Reader.


Vous pouvez également lire le guide de l'utilisateur en ligne (en anglais).

Le lecteur d'abord

  • Lire ebooks, BD et livres audio
  • Ajustez l'affichage
  • Support des lecteurs d'écran
  • Nombreuses options de navigation
  • Écouter via des voix de synthèse
  • Ajouter des catalogues
  • Plus de 20 langues

Annotations

  • Marquer les lieux de lecture
  • Surligner et annoter du texte dans EPUB
  • Sélectionner les styles d'annotation
  • Sélectionner le mode d'annotation
  • Export/import d'annotations

3.3.0 nouvelles fonctionnalités

  • Export des annotations en HTML
  • Filtrage des raccourcis
  • Options pour la voix de synthèse
  • Imprimer les PDF

Aidez-nous à développer Thorium Reader
