La aplicación de lectura
Para todos los lectores

  • Durante mucho tiempo, no hubo una aplicación de lectura moderna compatible con EPUB 3 que se pudiera utilizar en Windows, OSX y Linux, con acceso adecuado para personas con discapacidades de lectura, con un buen soporte para LCP DRM y capaz de explorar catálogos OPDS.
  • EDRLab decidió crear una aplicación de este tipo y lanzarla de forma gratuita, con el fin de ofrecer a los usuarios una excelente forma de disfrutar en pantalla grande de publicaciones EPUB, cómics/mangas/bandas dessinées, audiolibros y documentos PDF protegidos con LCP.
  • Las personas con discapacidad visual ahora se benefician de una aplicación de lectura de EPUB 3 que admite lectores de pantalla como Jaws y NVDA en Windows y Voice Over en Mac.

Tecnología

Thorium Reader se basa en un conjunto de módulos de código abierto sin navegadores, un conjunto de herramientas llamado Readium Desktop. Tanto Thorium Reader como Readium Desktop se basan en Electron.js, node.js y typescript, que son tecnologías multiplataforma eficientes. Thorium también utiliza componentes React basados en HTML 5/CSS 3.
